## üìù Customization Points

### Change Glow Intensity
In `styles.css`, adjust these variables:
```css
--glow-cyan-sm: 0 0 20px rgba(0, 217, 255, 0.3);
--glow-cyan-md: 0 0 30px rgba(0, 217, 255, 0.4);
--glow-cyan-lg: 0 0 40px rgba(0, 217, 255, 0.5);
```

### Change Glass Effect
```css
--glass-background: rgba(30, 64, 175, 0.1); /* Adjust transparency */
--glass-blur: blur(10px); /* Adjust blur amount */
```

### Change Accent Color
```css
--cyan-bright: #00D9FF; /* Your cyan color */
```
